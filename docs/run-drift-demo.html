5-Minute Drift Demo

This demo shows the core value of IAP: detect silent state mutation.

  1. Start from a clean virtual environment.
  2. Install the SDK and AMCS adapter.
  3. Generate an initial deterministic state root.
  4. Verify baseline is valid.
  5. Mutate state history and verify drift is detected.
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install iap-agent
python -m pip install -e /path/to/AMCS-0.1
python /path/to/iap-sdk/examples/state-drift-demo/demo.py

Expected terminal result includes:

verify_before_ok=True
verify_after_ok=False

The first check confirms continuity before mutation. The second check fails after state tampering, proving detectable drift.
